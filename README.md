# メルカリ売り上げ集計

メルカリの sold CSV、残高履歴、銀行明細、メルカード明細をもとに、税理士へ提出しやすいExcel資料を作るローカル実行ツールです。

目的は、売上を単純に合計することではありません。商品が売れた日、売上金が確定した日、メルカリ残高に加算された日、銀行振込・メルカード・メルペイなどで残高が減った流れを説明できる資料にすることです。

## 使い方

資料を `input` フォルダに入れてから、次を実行します。

```powershell
python src/main.py
```

出力先は `output` フォルダです。

```text
税理士提出用_メルカリ会計資料_日時.xlsx
```

## 今できること

- sold CSVから売上ベース、売上金確定ベース、月次集計を作成
- 販売利益をメルカリ残高の増加として残高消込表へ反映
- メルカリ残高履歴CSVを反映
- ゆうちょ銀行明細CSVを反映
- メルカードのExcel明細を反映
- 全額ポイント払いが混ざる可能性のあるメルカード明細は、集計から除外して未判定シートへ分離
- 不足資料チェックシートを作成
